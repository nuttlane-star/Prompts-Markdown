# Prompt Library (Markdown)

## Purpose

This repo is a place for me to store and refine reusable prompt elements and full prompts in Markdown format. It helps me avoid repeating the same context, constraints, and instructions as I work more with GenAI.

## What’s in here

- Reusable prompt building blocks (context, constraints, style, output formats)
- Complete prompts I’ve used and want to reuse or iterate on
- Notes on what worked well or poorly for specific prompts

## Focus areas

Most of my prompts are about payments technology, where I often need to develop and test positions and opinions. I also capture prompts for non-work topics, such as:

- Off-grid solutions
- Structural and engineering considerations for buildings
- Welding practices and techniques

## How I use this repo

- Capture new prompts as Markdown files.
- Refine prompts over time based on results.
- Reuse and adapt good prompts across different tools and use cases.
